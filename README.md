# ⚡ Energías en Argentina

Un proyecto web interactivo e informativo sobre los distintos tipos de energía (renovables y no renovables) y sus métodos de extracción en la República Argentina. 

El objetivo de este proyecto es educar y concientizar sobre la matriz energética del país, detallando cómo y dónde se genera la energía a través de una interfaz amigable y fácil de navegar.

## 🚀 Características Principales

- **Diseño Responsivo:** Creado con **Bootstrap** para garantizar que la web se adapte perfectamente a computadoras, tablets y celulares.
- **Modo Oscuro/Claro:** Incorpora un botón de accesibilidad para alternar entre el modo oscuro y claro de forma dinámica usando JavaScript.
- **Geolocalización:** Cada tipo de energía cuenta con enlaces directos a **Google Maps** para visualizar los puntos clave de generación o extracción en el país.
- **Marquesina Animada:** Una sección interactiva de "Trust Bar" con un carrusel infinito en CSS puro mostrando las principales empresas del sector.
- **Optimización SEO:** Estructura semántica básica aplicada para una mejor indexación.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica de todas las páginas y componentes.
- **CSS3:** Estilos personalizados, animaciones (keyframes) para la marquesina y manejo del Modo Oscuro a través de variables CSS.
- **JavaScript (Vanilla JS):** Lógica ligera encargada del estado y guardado (en `localStorage`) del tema oscuro.
- **Bootstrap 5:** Sistema de grillas, utilidades de Flexbox y diseño en tarjetas (Cards).

## 📁 Estructura del Proyecto

- `index.html` - Página principal con el resumen de la matriz energética y la barra de empresas.
- `[tipo-de-energia].html` - Páginas individuales detallando la energía Solar, Eólica, Fósil, Hidroeléctrica, Nuclear, Biomasa, Biogás, etc.
- `style.css` - Hoja de estilos principal con los tokens de diseño y animaciones.
- `script.js` - Controlador del tema claro/oscuro.
